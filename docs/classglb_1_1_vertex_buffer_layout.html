Detailed Description

Defines the layout of a vertex buffer, specifying the format of the vertex attributes.

The VertexBufferLayout class manages the layout of vertex data within a buffer. It allows you to specify the format of vertex attributes (such as position, color, etc.) and compute the stride required to traverse through the vertex data.
