Detailed Description

Provides an interface for rendering operations in OpenGL applications.

The Renderer class offers methods for clearing the screen, drawing arrays and elements, and setting rendering parameters such as clear color and viewport size. It supports various drawing modes as defined by OpenGL.

Enum class for specifying the drawing mode.

This enumeration defines different rendering modes used in OpenGL for drawing primitives.

Enumerator
Points 

Draw points.

LineStrip 

Draw a connected group of line segments.

LineLoop 

Draw a connected group of line segments, closing the loop.

Lines 

Draw separate line segments.

LineStripAdjacency 

Draw a connected group of line segments with adjacency information.

LinesAdjacency 

Draw separate line segments with adjacency information.

TriangleStrip 

Draw a connected group of triangles.

TriangleFan 

Draw a connected group of triangles sharing a common vertex.

Triangles 

Draw separate triangles.

TriangleStripAdjacency 

Draw a connected group of triangles with adjacency information.

TrianglesAdjacency 

Draw separate triangles with adjacency information.

Patches
