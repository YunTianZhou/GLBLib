Detailed Description

Provides functionalities for handling and querying cursor interactions.

The Cursor class manages cursor-related operations such as checking button states, retrieving cursor positions, and setting or disabling callbacks for cursor events. This class interacts with the GLFW library to handle mouse input and cursor events.
